# 05-03: Student Attendance Pages

## Objective

Create the attendance overview page (all students' stats) and individual student attendance detail page.

## Tasks

### Task 1: Attendance overview page

<read_first>
- src/app/(protected)/dashboard/page.tsx (existing pattern)
- src/app/api/courses/[courseId]/attendance-summary/route.ts (from 05-01)
</read_first>

<action>
Create `src/app/(protected)/dashboard/courses/[courseId]/attendance/page.tsx`:

1. "use client" component
2. Fetch attendance summary from `/api/courses/[courseId]/attendance-summary` on mount
3. Display:
   - Header: course name, back link to dashboard
   - Table with columns: 学号, 姓名, 出勤率 (colored percentage bar), 已签次数, 缺勤次数, 操作 (查看详情 button)
   - Support sorting by clicking column headers (学号, 出勤率, etc.)
   - Support search/filter by student name or ID (local filter)
   - When course has no sessions yet, show "暂无签到记录，签到后才会有出勤数据"
4. Click "查看详情" → navigate to student attendance detail page
5. Empty state: appropriate message based on whether course has any sessions
6. Loading state: spinner
7. Error state: error with retry
</action>

<acceptance_criteria>
- Table shows all students with attendance stats
- Color-coded attendance rate (green ≥80%, yellow ≥50%, red <50%)
- Column sorting works
- Search by name/ID works
- "查看详情" navigates to student detail
- Empty/loading/error states
- TypeScript: no type errors
</acceptance_criteria>
